hud:client:UpdateHud\nUpdate HUD display\n```lua\nTriggerEvent('hud:client:UpdateHud', hudData)\n```\n\n#### hud:client:SetTalking\nSet talking indicator\n```lua\nTriggerEvent('hud:client:SetTalking', true)\n```\n\n#### hud:client:SetSpeaker\nSet speaker indicator\n```lua\nTriggerEvent('hud:client:SetSpeaker', true)\n```\n\n#### hud:client:SetRecording\nSet recording indicator\n```lua\nTriggerEvent('hud:client:SetRecording', true)\n```\n\n#### hud:client:SetHandcuffed\nSet handcuffed indicator\n```lua\nTriggerEvent('hud:client:SetHandcuffed', true)\n```\n\n### Server Events\n\n#### hud:server:BroadcastUpdate\nBroadcast HUD update to all players\n```lua\nTriggerClientEvent('hud:server:BroadcastUpdate', -1, hudData)\n```\n\n#### hud:server:UpdatePlayerHud\nUpdate specific player HUD\n```lua\nTriggerClientEvent('hud:server:UpdatePlayerHud', playerId, hudData)\n```\n\n## File Structure\n\n```\njaneehud/\n‚îú‚îÄ‚îÄ
